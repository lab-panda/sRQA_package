## sRQA

An R package for symbolic recurrence quantification analysis of time series data.

## Installation

In RStudio: Packages → Install → Install from: Package Archive File, then browse to the downloaded file.

Or from the R console:
```r
install.packages("path/to/sRQA_0.3.0.tar.gz", repos = NULL, type = "source")
```

## Case Studies

| Folder | Description |
|--------|-------------|
| `case1` | Simulated data with known dynamical properties |
| `case2` | ECG: atrial fibrillation vs. normal sinus rhythm |
| `case3` | fMRI: dorsal attention network during movie-viewing and rest |
| `case4` | Speech: pause dynamics in truthful and deceptive statements (forthcoming) |

Case studies correspond to examples in the accompanying preprint (DOI forthcoming).
