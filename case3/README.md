## Data Source

BOLD time series data from Healthy Brain Network were obtained from:
https://reprobrainchart.github.io/docs/get_data

## Files

| File | Description |
|------|-------------|
| `pull_all_ND.txt` | PowerShell script to retrieve resting-state and movie-watching `.1D` files from the cloned HBN_CPAC repository via DataLad |
| `nt_folders.csv` | List of participant IDs in repository with no diagnosis |
| `copy_1D_files.py` | Copy `.1D` files to a paired directory, retaining only subjects with both resting-state and movie conditions |
| `merge_label_1D.py` | Concatenate `.1D` files and label ROIs with Schaefer atlas via Nilearn |
| `dan_glassbrains.py` | Glass brain visualizations of Dorsal Attention Network A and B subnetworks |
| `fMRI_pipeline.R` | Main analysis: symbolic RQA, cross-RQA, mixed-effects models, and coefficient plots |

Note: The preprocessed datafile generated by `merge_label_1D.py` exceeds GitHub's file size limit and is not included in this repository. Follow steps 1–4 to reproduce it.

## Reproducing the Results

1. Clone the HBN_CPAC repository from Reproducible Brain Charts.
2. Run `pull_all_ND.txt` to pull files from HBN_CPAC. Requires `nt_folders.csv`.
3. Run `copy_1D_files.py` to copy files to a paired directory for analysis. Requires `nt_folders.csv`.
4. Run `merge_label_1D.py` to concatenate files, label ROIs, and create the datafile for the pipeline.
5. Run `dan_glassbrains.py` to generate glass brain figures for DAN-A and DAN-B subnetworks (independent of previous steps).
6. Run `fMRI_pipeline.R` — Part 1 imports and cleans the data, Part 2 runs symbolic RQA and CRQA, Part 3 fits mixed-effects models with FDR correction, and Part 4 generates coefficient plots.

All file paths are hardcoded and will need to be updated to match your local directory structure.
